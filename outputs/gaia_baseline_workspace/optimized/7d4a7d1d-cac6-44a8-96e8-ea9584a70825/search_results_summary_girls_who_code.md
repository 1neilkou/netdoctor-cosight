# Search Results Summary: Girls Who Code - Women in Computer Science Statistics

## Source: Girls Who Code Official Website (https://girlswhocode.com/about-us)

### Key Data Points (directly from the page):
- **1995**: 37% of computer scientists were women
- **2017**: 24% of computer scientists were women  
- **2022**: 22% of computer scientists were women

### Direct Quote from the page:
"In 1995, 37% of computer scientists were women. Today, it's only 24%. The percent will continue to decline if we do nothing."

### Timeline visualization from the page:
- 1995 → 37%
- 2017 → 24%
- 2022 → 22%

### Analysis:
- Starting point: 37% (1995)
- Change of 13 percentage points: 37% - 13% = 24%
- Year when it reached 24%: 2017
- Time elapsed: 2017 - 1995 = **22 years**
